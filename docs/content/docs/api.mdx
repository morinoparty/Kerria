Kerriaは、外部プラグインから経済機能を利用するための2つの方法を提供しています。

## ServiceManager経由のAPI

Kerriaは起動時に `KerriaAPI` をBukkitの `ServiceManager` に登録します。
他のプラグインから以下のようにアクセスできます。

```kotlin
val registration = server.servicesManager.getRegistration(KerriaAPI::class.java)
val kerriaAPI = registration?.provider ?: error("Kerria is not loaded")

// アカウント管理
val accountManager = kerriaAPI.getAccountManager()
val account = accountManager.getOrCreateAccount(playerUUID, "PlayerName")

// 残高取得
val balance = accountManager.getBalance(accountId, currencyId)

// 入金・出金
val economyManager = kerriaAPI.getEconomyManager()
economyManager.deposit(accountId, currencyId, BigDecimal("100.00"))
economyManager.withdraw(accountId, currencyId, BigDecimal("50.00"))

// 通貨管理
val currencyManager = kerriaAPI.getCurrencyManager()
val currencies = currencyManager.getAllCurrencies()

// 取引ログ
val logManager = kerriaAPI.getLogManager()
val history = logManager.getTransactionHistory(accountId, limit = 10)
```

:::note
すべてのAPIメソッドは `Either<KerriaError, T>` を返します。
エラーハンドリングには Arrow の `fold`、`getOrNull`、`onLeft` などを使用してください。
:::

## Vault Economy API

Kerriaは [Vault](https://github.com/MilkBowl/Vault) Economy APIを実装しています。
Vault経由でアクセスする場合、操作はすべてデフォルト通貨に対して行われます。

```java
Economy economy = server.getServicesManager()
    .getRegistration(Economy.class).getProvider();

double balance = economy.getBalance(player);
economy.depositPlayer(player, 100.0);
economy.withdrawPlayer(player, 50.0);
```

:::info
Vault経由の取引では、呼び出し元プラグインが自動的に検出され、取引ログに記録されます。
:::

## カスタムイベント

Kerriaは経済操作が実行される前に `KerriaTransactionEvent` を発火します。
このイベントはキャンセル可能で、他のプラグインが取引を監視・制御できます。

```kotlin
@EventHandler
fun onTransaction(event: KerriaTransactionEvent) {
    // 取引の種類を確認
    when (event.type) {
        TransactionType.DEPOSIT -> { /* 入金 */ }
        TransactionType.WITHDRAW -> { /* 出金 */ }
        TransactionType.TRANSFER -> { /* 送金 */ }
        TransactionType.SET_BALANCE -> { /* 残高設定 */ }
    }

    // 取引の詳細
    val from = event.fromAccountId
    val to = event.toAccountId
    val amount = event.amount
    val currency = event.currencyId
    val caller = event.callerPluginName

    // 必要に応じてキャンセル
    if (shouldBlock(event)) {
        event.isCancelled = true
    }
}
```

### イベントプロパティ

| プロパティ | 型 | 説明 |
|-----------|-----|------|
| `type` | `TransactionType` | 取引の種類（DEPOSIT / WITHDRAW / TRANSFER / SET_BALANCE） |
| `fromAccountId` | `UUID` | 送金元アカウントID |
| `toAccountId` | `UUID` | 送金先アカウントID |
| `currencyId` | `Int` | 通貨ID |
| `amount` | `BigDecimal` | 取引金額 |
| `callerPluginName` | `String?` | 操作を実行したプラグイン名 |
| `isCancelled` | `Boolean` | キャンセル可能 |
