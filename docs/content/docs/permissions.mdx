Kerriaのコマンドはパーミッションノードで制御されます。

## プレイヤー向け

| パーミッション | 説明 | デフォルト |
|--------------|------|-----------|
| `kerria.balance` | 自分の残高を確認する | 全員 |
| `kerria.pay` | 他のプレイヤーに送金する | 全員 |
| `kerria.top` | 残高ランキングを表示する | 全員 |
| `kerria.log` | 自分の取引履歴を表示する | 全員 |

## 管理者向け

| パーミッション | 説明 |
|--------------|------|
| `kerria.admin.economy` | 残高の設定・付与・徴収（set / give / take） |
| `kerria.admin.currency` | 通貨の作成・削除・一覧・詳細表示 |
| `kerria.admin.log` | 他のプレイヤーの取引履歴を表示する |
