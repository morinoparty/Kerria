# データベース

KerriaはSQLiteとPostgreSQLをサポートしています。

## Table definition

### ACCOUNTS

プレイヤーのアカウント情報を管理するテーブルです。

<table>
    <thead>
        <tr>
            <th>column name</th>
            <th>type</th>
            <th>example</th>
            <th>description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>id</td>
            <td>uuid v7</td>
            <td><UUIDV7 /></td>
            <td>主キー Kerria用のアカウントID</td>
        </tr>
        <tr>
            <td>player_unique_id</td>
            <td>varchar(36)</td>
            <td><UUIDV4 /></td>
            <td>MinecraftプレイヤーのUUID（一意インデックス）</td>
        </tr>
        <tr>
            <td>player_name</td>
            <td>varchar(16)</td>
            <td>Notch</td>
            <td>プレイヤーの名前</td>
        </tr>
    </tbody>
</table>

### ACCOUNT_BALANCES

アカウントごとの通貨別残高を管理するテーブルです。1つのアカウントに対して複数の通貨の残高を保持できます。

<table>
    <thead>
        <tr>
            <th>column name</th>
            <th>type</th>
            <th>example</th>
            <th>description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>id</td>
            <td>bigint</td>
            <td>1</td>
            <td>主キー（自動採番）</td>
        </tr>
        <tr>
            <td>account_id</td>
            <td>uuid v7</td>
            <td><UUIDV7 /></td>
            <td>外部キー → ACCOUNTS(id)</td>
        </tr>
        <tr>
            <td>currency_id</td>
            <td>int</td>
            <td>1</td>
            <td>外部キー → CURRENCIES(id)</td>
        </tr>
        <tr>
            <td>balance</td>
            <td>decimal(20, 4)</td>
            <td>1,000,000</td>
            <td>残高</td>
        </tr>
    </tbody>
</table>

:::info 複合一意インデックス
`(account_id, currency_id)` の組み合わせに対して一意インデックスが設定されています。
1つのアカウントにつき、同じ通貨の残高レコードは1つだけ存在します。
:::

## ER図

```mermaid
erDiagram
    ACCOUNTS {
        uuid id PK
        varchar player_unique_id UK
        varchar player_name
    }

    ACCOUNT_BALANCES {
        bigint id PK
        uuid account_id FK
        int currency_id FK
        decimal balance
    }

    CURRENCIES {
        int id PK
        varchar name
        varchar plural
        varchar sign
        varchar format
        int fractional_digits
    }

    TRANSACTION_LOGS {
        bigint id PK
        uuid from_account_id FK
        uuid to_account_id FK
        int currency_id FK
        decimal amount
        datetime timestamp
    }

    ACCOUNTS ||--o{ ACCOUNT_BALANCES : "has"
    CURRENCIES ||--o{ ACCOUNT_BALANCES : "denominated in"
    ACCOUNTS ||--o{ TRANSACTION_LOGS : "sends"
    ACCOUNTS ||--o{ TRANSACTION_LOGS : "receives"
    CURRENCIES ||--o{ TRANSACTION_LOGS : "denominated in"
```
